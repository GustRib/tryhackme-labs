# TryHackMe Labs - Red Team Studies

Este repositório contém anotações e write-ups sem flags
baseados em laboratórios da plataforma TryHackMe.

Objetivo:
- Consolidar metodologia de pentest
- Praticar enumeração, exploração e pós-exploração
- Preparação para certificações iniciais (eJPT)

Todo o conteúdo segue princípios éticos e respeita o escopo dos laboratórios.
